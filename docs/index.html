Design Philosophy

Engineered Like Biology

Optimus mirrors the mechanisms that make living systems resilient.

🧬

Cell Division

→ delegate_task

One agent splits into many specialized workers. Each carries the right context for its job.

🧬

DNA

→ Role + Skill Templates

Roles define identity; skills define capability. Inherited, evolved, and recombined over time.

🛡️

Immune System

→ Quarantine + Auto-Recovery

Failing agents are isolated before they spread damage. Healthy replacements are spawned automatically.

🔥

Metabolism

→ Meta-Cron Engine

Background processes keep the system alive: scheduled audits, cleanup, and health monitoring.

Get Started

Three Commands to Orchestrate

Works with Claude Code, VS Code Copilot, Cursor, Windsurf, and any MCP client.

# 1. Initialize your workspace
$ npx -y github:cloga/optimus-code init

# 2. Configure your MCP client (auto-done for VS Code)
$ claude mcp add optimus-swarm -- npx -y github:cloga/optimus-code serve

# 3. Start orchestrating
$ "Delegate to the PM to build a user auth system"
